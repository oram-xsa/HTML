Manage Cluster

Join command for new node

Run this on the new machine to join it to this cluster:

curl -sfL https://get.rke2.io | INSTALL_RKE2_TYPE=agent sh - && echo "server: https://10.0.0.10:9345\ntoken: K10abc123.xxxxxxxxxx" > /etc/rancher/rke2/config.yaml && systemctl enable --now rke2-agent

Server nodes require odd total count (3, 5, 7) for quorum

The node will appear with this name once it connects

Warning: Removing a node will drain workloads first. This may cause brief service interruption if no other nodes can accept the pods.

Control plane endpoint of the target cluster (port 9345)

From /var/lib/rancher/rke2/server/node-token

Warning: This will remove the node from its current cluster and join a different one. All local cluster state will be reset.
